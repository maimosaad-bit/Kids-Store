# Kids Store React Project

This is a simple e-commerce React project for a kids clothing store.

## Features

- Browse products by categories
- Add products to cart
- Responsive design using React and CSS
- Navigation with React Router

## Tech Stack

- React
- React Router
- CSS / Tailwind (optional)
- JavaScript (ES6+)
